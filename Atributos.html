ATRIBUTOS HTML5

No. Elemento Definicion
1. Contexteditable Indica si el contenido del elemento es editable.
2. Contextmenu Define el ID del elemento que servira como menu de contexto para otro elemento
3. data-* Permite asociar atributos personalizados a un elemento HTML
4. Dir Define la direccion del texto. Los valores permitidos son Itr (Izq a Der) o rtl (Der a Izq)
5. Draggable Define si el elemento puede ser arrastrado
6. hidden Evita la prestacion del elemento dado, manteniendo los elementos secundarios
7. ID Identificador el elemento para acceder al mismo desde CSS o JavaScript debe de ser unico para cada elemento
8. lang Define el lenguaje utilizado en el elemento
9. language Define el lenguaje de programacion utilizado en el elemento
10. style Define los estilos CSS que anulan los estilos establecidos previamente
